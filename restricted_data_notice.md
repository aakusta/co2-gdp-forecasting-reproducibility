# Restricted Data Notice

This project uses public macroeconomic and energy data from World Bank sources together with carbon-pricing status information derived partly from ICAP source material.

ICAP-derived raw, cleaned, merged, and imputed files are not publicly redistributed in this repository because they contain third-party source material and derived values based on that material.

For reproducibility, this repository provides:

- source descriptions,
- variable definitions,
- processing notes,
- code structure,
- public-source data components where redistribution is permitted,
- aggregate manuscript outputs where redistribution is appropriate,
- and documentation indicating how restricted-source variables enter the analysis.

Researchers seeking full reproduction of the policy-variable results should obtain the relevant ICAP source material directly from ICAP according to ICAP’s access and redistribution terms.

Restricted ICAP-derived files were made available to the editor and reviewers for confidential peer-review purposes.

The public repository provides aggregate outputs and documentation aligned with the final revised manuscript; confidential reviewer materials contain the restricted ICAP-derived inputs required for full review reproduction.
